This project is a shooting game where the player has to protect Earth from 100 astero√Æds by clicking on them.

### Linux Installer

Clone the repository:

    git clone https://github.com/MayaHill/MyHunter-ShootingGame-15-29-November.git

Install CSFML

    sudo apt-get install libcsfml-dev libcsfml-doc

To compile:

    make

Update:

    git pull && make re

## How to play

    ./my_hunter -h

## Launch the game

    ./my_hunter
